# Network Traffic Analyzer

A simple command-line tool to capture and analyze network traffic in real-time.

## Features

-   List available network interfaces.
-   Capture live packets from a selected interface.
-   Display packet details (source/destination IP, protocol, etc.).
-   Show real-time statistics of the captured traffic.

## Setup

1.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the application:**
    ```bash
    python analyzer.py
    ```
